# Prova de conceito projetos transverssais 2
Esse é o firmware que será utilizado na prova de conceito da disciplina de projetos transverssais 2.
O projeto consiste na criação de um sensor de presença que envia notificações para o usuário quando
uma presença é detectada. 

## Etapa 1
Será usado um broker mqtt para receber os publishes do sensor de presença

## Etapa final
Usar AWS SNS para enviar notificações para o proprietário da casa
